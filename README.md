# Tybo
A custom admin engine for Ruby on Rails

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'tybo'
```

And
```bash
$ bundle install
```
Then execute the generator
```bash
$ bundle exec rails g tybo_install
```
